\emph{\textbf{Keywords:} } IoT, FreeSotfware, OpenSource, Science, Technology, GPS, Bluetooth, WIFI, Smarth Glass, EPSON.

\begin{center}
\textbf{Resumen}
\end{center}

El presente escrito presenta de manera ordenada, precisa y entendible el desarrollo de un sistema con el cual diverso tipo de usuarios de una empresa, colegio, centro de salud u otra entidad puedan ingresar u obtener acceso a diferentes servicios presentando su tarjeta o llave de radiofrecuencia, en esta sección solo se presenta el diseño del sistema de hardware y firmware, como a su vez debe considerarse este escrito como un recetario técnico de dicho sistema.
